# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2025-11-17
### Added
- Initial project setup with Express, TypeScript, Prisma, and PostgreSQL
- CRUD endpoints for customers
- Advanced query filters (channel, tier, gender, flavors, date ranges)
- Zod validation for create and update
- Soft delete functionality
- Modular filter utility and validator layer
- Clean `.gitignore`, `README.md`, `.env.example`, and `CONTRIBUTING.md`
