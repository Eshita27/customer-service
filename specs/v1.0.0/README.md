# Swagger Spec v1.0.0

This folder contains the OpenAPI specification for version `v1.0.0` of the Customer Service API.

## Files
- `openapi.yaml`: YAML format for Swagger UI and Redoc
- `openapi.json`: JSON format for tooling and validators
